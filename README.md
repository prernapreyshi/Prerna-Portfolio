# Prerna Portfolio - Personal Portfolio Website

**Prerna Portfolio** is a personal portfolio website designed to showcase the skills, projects, and achievements of **Prerna**, a passionate web developer. The website highlights various aspects of the developer's journey, including her technical expertise, the projects she has worked on, and her contact information.

## Features:
- **Home Page**: A brief introduction to who Prerna is, highlighting her skills, experience, and areas of expertise.
- **About Me Page**: A detailed section providing more about Prerna's background, passion for web development, and career journey.
- **Projects Page**: A collection of projects that Prerna has worked on, including project descriptions, links to live demos, and GitHub repositories.
- **Skills Page**: A section showcasing Prerna's technical skills, tools, and technologies she is proficient in.
- **Contact Page**: A form that allows visitors to get in touch with Prerna, along with links to her social media profiles and professional platforms.
- **Responsive Design**: The portfolio is designed to be mobile-friendly, ensuring it looks great on all devices, including smartphones and tablets.

## Technology Stack:
- **HTML**: Provides the structure for the website.
- **CSS**: Used for styling the website, ensuring it's visually appealing and responsive.
- **JavaScript**: Adds interactive features such as form validation, smooth scrolling, and dynamic content loading.
- **GitHub Pages**: Hosting the website to make it publicly accessible to visitors.
